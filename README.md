# Мультимодальная нейросеть для предсказания калорийности блюд

В этом проекте создана и обучена мультимодельная нейросеть предсказания калорийности блюда по его фотографии и краткому описанию (ингредиенты, вес). Полученную модель можно интегрировать во многие fitness и health-приложения, она будет полезна тем, кто следит за своим здоровьем.
